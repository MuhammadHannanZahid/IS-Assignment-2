> A modern, single-page web application for secure text encryption, decryption, and cryptographic hashing. Built with a responsive **Glassmorphism UI** and powered by a secure **Python/Flask backend**.

---

## Features

- **AES-256 Encryption:** Industry-standard symmetric encryption using Cipher Block Chaining (CBC) and randomized Initialization Vectors (IV).
- **Dynamic Caesar Cipher:** Classic substitution cipher with a dynamically adjustable shift value that updates the UI in real-time.
- **SHA-256 Hashing:** Secure, one-way cryptographic hashing for data integrity verification.
- **Base64 Encoding:** Quick binary-to-text encoding and decoding.
- **Modern UI/UX:** A sleek, responsive interface featuring gradient backgrounds, glassmorphism cards, and interactive feedback.
- **One-Click Copy:** Easily copy ciphertexts or plaintexts to your clipboard with visual confirmation.

---

## Tech Stack

### Frontend
- **HTML5:** Semantic structure.
- **CSS3:** Custom styling, flexbox layouts, and glassmorphism design.
- **JavaScript (Vanilla):** DOM manipulation, event handling, and asynchronous API calls using the `Fetch API`.

### Backend
- **Python:** Core backend logic.
- **Flask:** Lightweight web framework to serve the frontend and handle RESTful API requests.
- **PyCryptodome:** Cryptographic library for secure AES operations.
- **Hashlib / Base64:** Built-in Python libraries for hashing and encoding.
